📌 Project Overview

Uber-Data-Analysis-Dashboard-using-power-bi is a Business Intelligence project developed in Power BI to analyze Uber ride data and generate actionable insights.

The dashboard helps understand ride patterns, revenue trends, peak booking times, customer behavior, and operational performance using interactive visuals and KPIs.

🎯 Project Objectives

Analyze Uber ride booking data

Identify peak hours and high-demand locations

Track revenue and trip performance

Understand customer and driver trends

Support data-driven decision-making

🛠️ Tools & Technologies Used

Power BI Desktop

Power Query (Data Cleaning & Transformation)

DAX (Data Analysis Expressions)

Microsoft Excel / CSV Dataset
